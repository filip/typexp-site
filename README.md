# Typexp

**Menu bar text expansion for macOS.**

Type a short trigger, press space — Typexp replaces it with your full text instantly, in any app. Supports fill-in `{{variables}}` for dynamic content.

[typexp.app](https://typexp.app) · [Download](https://github.com/filip/typexp-site/releases/latest/download/Typexp.zip)

## Features

- System-wide expansion — works in Mail, Slack, Notion, browsers, terminals, anywhere
- `{{variable}}` placeholders with a floating fill-in panel
- Recent snippets in the menu bar
- iCloud sync across all your Macs
- Automatic weekly backups
- 7-day free trial · £5.99 one-time purchase

## Requirements

- macOS 14 Sonoma or later
- Accessibility permission (the app guides you through it on first launch)

## Installation

1. [Download Typexp](https://github.com/filip/typexp-site/releases/latest/download/Typexp.zip)
2. Unzip and move `Typexp.app` to your Applications folder
3. Launch and grant Accessibility permission when prompted
